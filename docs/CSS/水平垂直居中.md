```css
.parent{
  background-color: aquamarine;
  width: 500px;
  height: 500px;
  position: relative;
}
.child {
  position: absolute;
  transform: translate(-50%, -50%);
  left: 50%;
  top: 50%;
  width: 100px;
  height: 100px;
  background-color: aqua;
}
```
