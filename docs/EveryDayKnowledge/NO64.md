## 箭头函数的 this 打印出来的值永远都一样吗

- 不一样，在定义的时候函数的 this 是可以改变的

```js
function f() {
  const f1 = () => {
    console.log(this);
  };
  f1();
}
f();
f.call({});
```

## ES6类转es5
```js
class A{
  a(){
    console.log('a')
  }
}
A.prototype.b = function(){}

let a = new A()
for(let key in a) {
  console.log(key)
}
// b
```

在babel转es5时会设置属性描述符，会将**可枚举属性**置为false,所以此时遍历不到a属性
