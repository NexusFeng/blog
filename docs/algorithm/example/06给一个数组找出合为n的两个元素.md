## 题目描述

- 有一个递增的数组[1,2,4,7,11,15]和一个 n = 15
- 数组中有两个数,和是 n,即 4+11=15
- 写一个 js 函数,找出这两个数

## 思路

常规思路,时间复杂度为 O(n^2)

- 嵌套循环,找出一个数,然后去遍历下一个数,求和,判断
- 时间复杂度 O(n^2)，不可用

双指针思路,时间复杂度降低为 O(n)

- 随便找两个数(头尾)
- 如果和大于 n,则需要向前寻找
- 如果和小于 n，则需要向后寻找 - 二分法思想

```ts
function findTwoNumber(arr: number[], n: number): number[] {
  const res: number[] = []

  const length = arr.length
  if (length === 0) return res

  let i = 0 //头
  let j = length - 1 //尾
  while (i < j) {
    const n1 = arr[i]
    const n2 = arr[j]
    const sum = n1 + n2
    if (sum > n) {
      j--
    } else if (sum > n) {
      i++
    } else {
      res.push(n1, n2)
      break
    }
  }
  return res
}
```

## Jest 单元测试

```ts
// findTwoNumber.test.ts
import findTwoNumber from "./"
describe("查找和为n的两个数", () => {
  it("正常情况", () => {
    const arr = [1, 2, 4, 7, 11, 15]
    const res = findTwoNumber(arr, 15)
    expect(res).toEqual([4, 11])
  })
  it("空数组", () => {
    const res = findTwoNumber([], 15)
    expect(res).toEqual([])
  })
  it("找不到结果", () => {
    const arr = [1, 2, 4, 7, 11, 15]
    const res = findTwoNumber(arr, 100)
    expect(res).toEqual([])
  })
})
```
