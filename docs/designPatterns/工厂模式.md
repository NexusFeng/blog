## 设计模式的核心
观察整个逻辑里面的变与不变,然后将变与不变分离,达到使变化的部分灵活、不变的部分稳定的目的。
## 概念
工厂模式其实就是**将创建对象的过程单独封装**

## 应用场景
在有构造函数的地方

## 简单工厂
```js
function User(name, age, career, work) {
  this.name = name
  this.age = age
  this.career = career
  this.work = work
}

function Factory(name, age, career) {
  let work
  switch (career) {
    case 'corder':
      work = ['写代码', '写系分', '修bug']
      break;
  
    case 'manager':
      work = ['订会议室', '写PRD', '见客户']
      break;
    // 其他情况
    // .....
  }

  return new User(name, age, career, work)
}
```
